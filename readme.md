### 🚀 **最新版本：v2.0.0** | **更新日期：2025年7月30日**

## ✨ **機器人特色**

**Sigma Bot** 是一個功能完整的 Discord 社群管理機器人，專為**大型社群**和**遊戲公會**設計，提供超過 **80+ 指令**和 **30+ 自動化功能**。

### 🌟 **核心亮點**

- 🧠 **AI 智能聊天** - 整合 OpenAI GPT 與網路搜尋功能
- 🛡️ **先進防護系統** - 防洗版、防突襲、防邀請、威脅偵測
- 🎵 **專業音樂系統** - 支援 YouTube、Spotify、SoundCloud 等平台
- 🎰 **創新扭蛋系統** - 可自訂獎品的虛擬抽獎機制
- 🎁 **進階抽獎功能** - 條件式抽獎與自動化管理
- 📊 **等級排行系統** - 用戶活躍度追蹤與獎勵機制
- ⚙️ **統一設定面板** - 簡化的管理介面與一鍵設定

## 📋 **功能概覽**

### 🏘️ **社群功能 (18+ 指令)**
```
/help        - 互動式指令說明系統
/serverinfo  - 詳細伺服器資訊統計
/report      - 伺服器內檢舉系統
/suggest     - 建議與回饋收集
/fun         - 娛樂指令集合
/vote        - 投票與民調系統
```

### 🛡️ **管理功能 (25+ 指令)**
```
/ban /kick /mute     - 基礎管理指令
/warn /unwarn        - 警告系統
/purge               - 批量訊息清理 (最多300則)
/settings            - 統一設定系統 (7大類別)
/security            - 安全分析工具
/permissions         - 進階權限管理
```

### 🎵 **音樂系統 (12+ 指令)**
```
/play /pause /skip   - 播放控制
/queue /shuffle      - 佇列管理
/volume /loop        - 音效設定
支援平台: YouTube, Spotify, SoundCloud, Deezer
```

### 🎰 **扭蛋系統 (全新功能)**
```
/gacha list          - 查看可用扭蛋機
/gacha play          - 進行扭蛋抽取
/gacha balance       - 查看扭蛋幣餘額
/gacha history       - 抽取歷史記錄
/gacha-admin         - 管理員扭蛋機管理
```

### 🎁 **抽獎向導 (智能系統)**
```
/giveaway            - 進階抽獎創建向導
- 條件設定: 身分組、等級、活躍度
- 外觀自訂: 顏色、圖片、描述
- 自動化管理: 開獎、通知、記錄
```

### 📊 **等級系統**
```
/level check         - 查看用戶等級
/level leaderboard   - 伺服器排行榜
/level settings      - 個人化設定
自動經驗值獲取: 訊息、語音、互動
```
